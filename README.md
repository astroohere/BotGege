# BotGege
A Simple WhatsApp bot by Astroo, made with javascript and baileys

```
how to install

1. copy the repo
git clone https://github.com/astroohere/BotGege.git

2. cd to folder
cd BotGege

3. install module
npm i

4. start bot
npm start
```
